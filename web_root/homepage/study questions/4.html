1. What are the five different position values?
    The five different position values are static, relative, fixed, absolute, and sticky.
  2. Which position do HTML elements have by default?
    HTML element are static by default.
  3. Explain how position: relative; works.
    An element with position: relative; is one that is positioned relative to its normal position.
  4. An element with position: fixed; is positioned relative to what?
    An element with position: fixed; is positioned relative to the viewport and will always stay in the same place even when the user scrolls up or down and zooms in and out.
  5. How is an element with position: absolute; positioned?
    An element with position: absolute; is positioned relative to that of the nearest positioned ancestor. If there is no specified positioned ancestor, it uses the document body instead.
  6. Does an element with fixed positioning scroll?
    No it does not.
  7. How does an element with position: sticky; work?
    An element with position: sticky; alternates between relative and fixed depending on the user's scroll position. It is positioned relative until the page is scrolled to a certain area of the viewport, where it will then toggle to position: fixed;
  8. A sticky element toggles between which two positioning properties?
    A sticky element toggles between relative and fixed.
  9. Which property specifies the stack order of an element (which element should be placed in front of, or behind, the others) when elements are overlapping?
    The Z-index property.
  10. Which CSS property controls what happens to content that is too big to fit into an area?
    The overflow property.
  11. Which overflow value creates a scrollbar in order to see the overflow content?
    The scroll property.
  12. What are the four float property values?
    The four float property values are left, right, none, and inherit.
  13. Which property allows you to wrap text around an image?
    The float property.
  14. Which property specifies what elements can float beside the cleared element and on which side?
    The clear property.
  15. Explain the Clearfix Hack.
    The clearfix hack is a command that will fix the issue of a floated element overflowing outside of its container by adjusting the container size to fit the overflowing element.
  16. What is the major difference between display: inline; and display: inline-block;?
    The main difference is that display: inline-block can have it's width and height adjusted, while display: inline values can not be adjusted. Another key difference is that display: inline-block respects top and bottom margins or paddings, while display: inline does not.
  17. What is one common use of display: inline-block;?
    One common use for display: inline-block is displaying a list of items horizontally instead of displaying them vertically.
  18. How do you horizontally center a block element (like <div>)?
    By using margin: auto;
  19. What are three ways to center an element vertically in CSS?
    Using padding for the top and bottom margins, making the line-height property value equal to that of the height property, and using the positioning and transform properties to align it yourself.
  20. What is a combinator?
    A combinator is used to explain the relationship between a group of selectors.
  21. What are the four different combinators?
    The four different combinators are the descendant selector, the child selector, the adjacent sibling selector, and the general sibling selector.
  22. What is a descendant selector?
    The descendant selector selects and matches all elements that are descendants of a specified element.
  23. What is a child selector?
    The child selector selects all of the elements that are the children of a specified element.
  24. What is an adjacent sibling selector?
    The adjacent sibling selector selects and element that directly follows after another specified element.
  25. What are Pseudo-classes?
    A pseudo-class is used to define a special state of an element.
  26. Give three examples of what a pseudo-class can be used for.
    A pseudo-class can be used to style an element whe a mouse hovers over it, give a different style to visited and unvisited links, and style an element when it comes into focus.
  27. Which pseudo-class allows you to style an <a> element when a cursor is over it?
    The a:hover pseudo-class.
  28. What are Pseudo-elements?
    A pseudo-element is used to style a specified part or parts of an element.
  29. What are the opacity property values and which is more transparent?
    The opacity property is what specifies the transparency of an element.
  30. The opacity property is often used together with the which selector to change the opacity on mouse-over?
    The :hover selector.
  31. When using the opacity property to add transparency to the background of an element, all of its child elements inherit the same transparency. What can happen to text inside a transparent element?
    It can become fully transparent alongside the background and no longer be readable.
  32. What is an image sprite and why are they used?
    An image sprite is a collection of images grouped together into a single image. It's mainly used to reduce the loading time and save bandwidth for a webpage.
  33. What is an attribute selector?
    The attribute selector is used to select elements that have a specified attribute.
  34. When expressing length in CSS, do you use whitespace between the number and the unit?
    No, you do not.
  35. What are the two types of length units?
    The two types of length units are absolute units and relative units.
  36. What are two common length units?
    in and cm.
  37. In CSS, are negative lengths are allowed?
    Only for some properties.
  38. What is specificity?
    Specificity is essentially a score that determines which selector in a group of selectors has the highest priority and will be applied to an element.
  39. Every selector has its place in the specificity hierarchy. What are the four categories which define the specificity level of a selector?
    The four categories that define the specificity level of a selector are Inline styles, IDs, Classes/pseudo-classes/attribute selectors, and elements/pseudo-elements.
  40. If the same rule is written twice into the external style sheet, which rule will be applied?
    The latest rule.
