1. What is responsive web design?
    Responsice web design is the process of making your webpage suitable for various screen sizes.
  2. Should web pages leave out information to fit smaller devices?
    In most cases, no. It should try to contain all of it's information if possible.
  3. Responsive web design is when you use CSS and HTML to do what?
    Adjust the content on a webpage to look good for different viewport sizes.
  4. What is the viewport?
    The visible area of a webpage.
  5. Before tablets and mobile phones, web pages were designed only for computer screens, and it was common for web pages to have what?
    A static design and a fixed size.
  6. HTML5 introduced a method to let web designers take control over the viewport, through which tag?
    The <meta> tag.
  7. If the user is forced to scroll horizontally, or zoom out, to see the whole web page it results in what?
    A poor user experience.
  8. Many web pages are based on a grid-view, which means what?
    The page is organized into columns.
  9. A responsive grid-view often has how many columns, and has a total width of what?
    It typically has 12 columns and a width of 100%.
  10. When building a responsive grid-view, why are all HTML elements set to have the box-sizing property set to border-box?
    Because it is necessary to ensure the padding and border measurements are taken into account when calculating the total width and height of elements.
  11. What is a media query?
    A rule that includes defined CSS properties if a certain condition is true.
  12. When were media queries introduced?
    In the release of CSS3.
  13. What is a breakpoint?
    The area that marks where certain elements will behave differently based on what size your screen is.
  14. If the width property is set to 100%, a responsive image will do what?
    The image becomes responsive and will scale alongside the viewport size.
  15. If a background-image has the background-size property is set to “contain,” what will happen to the background-image?
    It will scale to fit the content area while still maintaining it's aspect ratio.
  16. If the background-size property is set to “100% 100%,” what will happen to the background image?
    It will stretch to fill the page. Some of the image may be clipped depending on it's width and height.
  17. If the background-size property is set to “cover,” what will happen to the background image?
    It will cover the entire specified area.
  18. Can you set different background images for different devices?
    Yes. Using media queries gives us the ability to specify different images when a screen scales to a certain size.
  19. List two free responsive web design frameworks.
    W3.CSS and Bootstrap.
  20. Are the responsive web site templates at the W3Schools’ free to modify, save, share, and use in all your projects?
    Yes.
